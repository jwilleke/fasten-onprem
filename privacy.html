The short version

YourPHR is self-hosted. Your health records live on the server you run. The YourPHR project and this website (yourphr.org) do not receive, store, or have any access to your medical data.

This website (yourphr.org)

This is a static informational site hosted on GitHub Pages. It sets no cookies and runs no analytics or third-party trackers. GitHub may log standard request metadata (such as IP address) to serve the page; see GitHub's privacy documentation.

The YourPHR application

When you self-host YourPHR, all data — imported FHIR records, accounts, and credentials — stays within your own deployment. The project does not phone home. As the operator, you control storage, backups, encryption, and access.
